Audio plays for both local sig-file events and remote HTTP events; use Mute next to a sound to silence that event, or pick a different WAV file. HTTP listening on 127.0.0.1:9123. From a remote host (SSH-tunneled), POST to one of: 
curl -X POST http://localhost:9123/event/stop
curl -X POST http://localhost:9123/event/idle
curl -X POST http://localhost:9123/event/permission
curl -X POST http://localhost:9123/event/active   # dismiss alerts
Add this to your outbound SSH from Windows to the remote: ssh -R 9123:127.0.0.1:9123 user@remote-host
